Installation Instructions

1. Install the Font:
Download the BlissFont-Regular.otf file, open it on your computer, and click Install Font.
2. Set Up RIME Input Engine:
Download and install the RIME input engine for your operating system:
  • macOS: Squirrel
  • Windows: Weasel
  • Linux: ibus-rime or fcitx-rime
3. Copy Configuration Files:
Copy the downloaded blissymbols.dict.yaml, blissymbols.schema.yaml, and the OS-specific custom file into your RIME user directory:
  • macOS (Squirrel): Copy files to ~/Library/Rime/ and rename squirrel.custom.yaml to just squirrel.custom.yaml (or merge with existing).
  • Windows (Weasel): Copy files to %APPDATA%\Rime\ and rename weasel.custom.yaml to weasel.custom.yaml.
  • Linux (ibus-rime): Copy files to ~/.config/ibus/rime/.
4. Register and Deploy Schema:
Open the default.custom.yaml file in your RIME directory (create it if it does not exist) and register the schema under schema_list: 
patch:
  schema_list:
    - schema: blissymbols
Then, click Deploy from the RIME tray menu. You can now toggle to the Blissymbolics input method globally and type using the QWERTY keys or by typing English/Swedish gloss words (e.g. typing "heart" to select the Blissymbol heart).
